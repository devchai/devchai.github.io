제1조 (목적)

이 약관은 미래(이하 "사업자")가 토스 앱인토스 플랫폼을 통해 제공하는 정부지원금 미니앱 서비스(이하 "서비스")의 이용에 관한 사업자와 이용자 간의 권리, 의무 및 책임 사항을 규정하는 것을 목적으로 해요.

제2조 (용어의 정의)

이 약관에서 사용하는 용어의 뜻은 다음과 같아요.

  1. "서비스"란 사업자가 토스 앱인토스 미니앱으로 제공하는 정부지원금 정보 조회 서비스를 말해요.
  2. "이용자"란 이 약관에 따라 서비스를 이용하는 자를 말해요.
  3. "미니앱"이란 토스 앱 내부에서 WebView 기반으로 동작하는 경량 애플리케이션을 말해요.
  4. "북마크"란 이용자가 관심 있는 정부지원금 서비스를 기기에 저장하는 기능을 말해요.
  5. "공공데이터"란 정부24 공공데이터포털(api.odcloud.kr)에서 제공하는 정부지원 서비스 목록 데이터를 말해요.

제3조 (약관의 효력 및 변경)

  1. 이 약관은 서비스 화면에 게시하거나 기타 방법으로 이용자에게 공지함으로써 효력이 발생해요.
  2. 사업자는 관련 법령을 위배하지 않는 범위 내에서 이 약관을 변경할 수 있어요.
  3. 약관이 변경되는 경우, 변경 내용과 시행일을 서비스 내에서 최소 7일 전에 공지해요.
  4. 변경된 약관 시행 이후에도 서비스를 계속 이용하면 변경된 약관에 동의한 것으로 봐요.

제4조 (서비스의 내용)

사업자가 제공하는 서비스의 내용은 다음과 같아요.

기능 설명 유형
정부지원금 목록 조회 정부24 공공데이터 API를 통해 정부지원 서비스 목록을 조회하고 표시해요 기본
무한 스크롤 리스트 하단 도달 시 추가 데이터를 자동으로 불러와요 기본

제5조 (서비스 이용)

  1. 서비스는 토스 앱 내 앱인토스 플랫폼을 통해서만 이용할 수 있어요.
  2. 서비스 이용을 위해 별도의 회원 가입이나 로그인은 필요하지 않아요.
  3. 서비스는 무료로 제공하며, 별도의 요금이 발생하지 않아요.
  4. 서비스 이용 시 데이터 통신료는 이용자의 통신사 요금제에 따라 부과될 수 있어요.

제6조 (북마크 기능)

  1. 이용자는 관심 있는 정부지원금 서비스를 북마크에 추가하거나 제거할 수 있어요.
  2. 북마크 데이터는 이용자의 기기(브라우저 localStorage)에만 저장되며, 서버로 전송하지 않아요.
  3. 다음의 경우 북마크 데이터가 삭제될 수 있어요.
    • 브라우저 캐시/데이터 삭제
    • 앱 재설치
    • 기기 변경
    • 미니앱 데이터 초기화
  4. 사업자는 북마크 데이터의 영구 보존을 보장하지 않으며, 데이터 손실에 대한 책임을 지지 않아요.

북마크 데이터는 이용자의 기기에만 저장돼요. 기기 변경이나 데이터 삭제 시 복구할 수 없으니 참고해 주세요.

제7조 (검색 및 필터 기능)

  1. 이용자는 서비스명 키워드로 정부지원금을 검색할 수 있어요.
  2. 검색은 정부24 공공데이터 API의 서버 사이드 검색 기능을 활용하며, 검색 결과는 API 데이터 기준으로 제공해요.
  3. 필터 기능은 사용자구분, 부서명, 서비스분야, 지원유형 카테고리를 기준으로 이용할 수 있어요.
  4. 검색 및 필터 결과의 정확성과 완전성은 공공데이터 API의 데이터 품질에 따라 달라질 수 있어요.

제8조 (광고)

  1. 서비스는 운영 비용 충당을 위해 광고를 포함할 수 있어요.
  2. 광고 유형은 다음과 같아요.
    • 리워드 광고: 이용자가 광고 시청을 완료하면 보상이 제공되는 광고예요.
    • 전면 광고: 리워드 광고 로드 실패 시 대체로 노출되는 광고예요.
  3. 광고는 상세 페이지 진입 시 일정 빈도(6회 진입 중 1회)로 노출되며, 서비스 이용 흐름을 과도하게 방해하지 않아요.
  4. 광고는 Google AdMob을 통해 제공되며, 광고 콘텐츠에 대한 책임은 해당 광고주에게 있어요.
  5. 사업자는 광고의 내용, 정확성, 적법성에 대해 보증하지 않아요.

광고 콘텐츠는 사업자가 아닌 제3자(광고주)가 제공해요. 광고를 통한 거래나 피해에 대해 사업자는 책임을 지지 않아요.

제9조 (외부 콘텐츠 연결)

  1. 서비스는 정부24 웹사이트 등 외부 웹페이지로의 연결(바텀시트 모달)을 제공할 수 있어요.
  2. 외부 웹페이지의 콘텐츠는 해당 사이트 운영자가 관리하며, 사업자는 그 내용에 대해 책임을 지지 않아요.
  3. 외부 웹페이지 이용 시에는 해당 사이트의 이용 약관 및 개인정보처리방침이 적용돼요.
  4. 외부 웹페이지의 접근성, 가용성, 정확성에 대해 사업자는 보장하지 않아요.

제10조 (데이터 저장 및 개인정보)

10.1 기기 내 데이터 저장

서비스는 다음의 데이터를 이용자의 기기에 저장해요.

저장 위치 저장 항목 용도 보존 기간
localStorage 북마크된 서비스 정보 북마크 기능 제공 이용자 삭제 시까지
sessionStorage 상세 페이지 방문 횟수 광고 노출 빈도 관리 세션 종료 시 자동 삭제

10.2 개인정보

  1. 서비스는 이용자의 개인정보를 직접 수집하지 않아요.
  2. 서비스는 별도의 회원 가입, 로그인, 개인 식별 정보를 요구하지 않아요.
  3. 기기에 저장되는 데이터(북마크, 방문 횟수)는 개인을 식별할 수 없는 정보예요.
  4. 광고 서비스(Google AdMob) 이용에 따른 데이터 처리는 Google의 개인정보처리방침을 따라요.
  5. 토스 앱인토스 플랫폼 이용에 따른 데이터 처리는 비바리퍼블리카(토스)의 개인정보처리방침을 따라요.

10.3 외부 API 통신

  1. 서비스는 정부24 공공데이터 API(api.odcloud.kr)와 통신하여 정부지원금 정보를 조회해요.
  2. API 통신 시 이용자의 개인정보는 전송되지 않으며, 서비스 데이터 조회 목적으로만 사용해요.

제11조 (지식재산권)

  1. 서비스의 디자인, 코드, UI/UX 등 사업자가 창작한 콘텐츠에 대한 지식재산권은 사업자에게 있어요.
  2. 정부지원금 데이터는 공공데이터로서 공공누리 이용 조건에 따라 제공돼요.
  3. 토스 디자인 시스템(TDS) 관련 지식재산권은 비바리퍼블리카에 있어요.
  4. 이용자는 서비스를 통해 제공받은 정보를 개인적, 비상업적 목적으로만 이용할 수 있어요.

제12조 (면책 사항)

  1. 데이터 정확성: 서비스에서 제공하는 정부지원금 정보는 정부24 공공데이터 API를 통해 제공받은 것으로, 실제 지원 조건, 금액, 신청 자격은 해당 정부 기관에 직접 확인해야 해요.
  2. 서비스 가용성: 네트워크 환경, API 서버 상태, 플랫폼 점검 등으로 인해 서비스가 일시적으로 중단될 수 있으며, 이로 인한 손해에 대해 사업자는 책임을 지지 않아요.
  3. 광고 콘텐츠: 광고를 통해 제공되는 제품, 서비스, 정보에 대한 책임은 해당 광고주에게 있어요.
  4. 기기 데이터: 이용자 기기에 저장된 데이터(북마크 등)의 손실에 대해 사업자는 책임을 지지 않아요.
  5. 외부 서비스: 서비스를 통해 연결되는 외부 웹사이트(정부24 등)에서 발생하는 문제에 대해 사업자는 책임을 지지 않아요.
  6. 간접 손해: 서비스 이용과 관련하여 발생한 간접적, 부수적, 결과적 손해에 대해 사업자는 법률이 허용하는 최대 범위 내에서 책임을 제한해요.

정부지원금 신청 자격, 지원 금액, 신청 기한 등은 반드시 해당 정부 기관에 직접 확인해 주세요. 서비스에서 제공하는 정보는 참고용이에요.

제13조 (서비스 중단 및 변경)

  1. 사업자는 다음의 경우 서비스의 전부 또는 일부를 중단할 수 있어요.
    • 시스템 정기 점검 또는 긴급 보수
    • 정부24 공공데이터 API의 변경, 중단, 또는 폐지
    • 토스 앱인토스 플랫폼의 정책 변경 또는 서비스 중단
    • 천재지변, 국가 비상사태, 그 밖의 불가항력적 사유
  2. 사업자는 서비스의 기능을 추가, 변경, 또는 삭제할 수 있으며, 중요한 변경 사항은 사전에 공지해요.
  3. 서비스 종료 시, 최소 30일 전에 서비스 내에서 공지해요.

제14조 (이용자의 의무)

이용자는 서비스 이용 시 다음 사항을 준수해야 해요.

  1. 서비스를 정상적인 용도로만 이용해야 해요.
  2. 서비스의 운영을 방해하는 행위를 해서는 안 돼요.
  3. 서비스를 통해 제공받은 정보를 상업적 목적으로 무단 이용하거나 재배포해서는 안 돼요.
  4. 서비스의 기술적 보호 조치를 우회하거나 해제하는 행위를 해서는 안 돼요.
  5. 자동화 도구(봇, 스크래퍼 등)를 사용하여 서비스에 접근해서는 안 돼요.
  6. 관련 법령 및 이 약관의 규정을 준수해야 해요.

제15조 (분쟁 해결)

  1. 이 약관과 관련한 분쟁은 대한민국 법률을 준거법으로 해요.
  2. 서비스 이용과 관련하여 사업자와 이용자 간에 분쟁이 발생한 경우, 양측은 원만한 해결을 위해 성실히 협의해요.
  3. 협의가 이루어지지 않는 경우, 관할법원은 민사소송법에 따른 법원으로 해요.
  4. 이용자는 한국소비자원, 전자거래분쟁조정위원회 등 분쟁 조정 기관을 통해 분쟁 해결을 신청할 수 있어요.
